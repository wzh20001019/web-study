#### 1. git 基本配置

```txt
// 配置用户信息

git config --global user.name "用户名"
git config --global user.email "邮箱"
```

```txt
// 检查配置信息
git config --list --global

git config user.name
git config user.email
```

```txt
// 获取帮助信息

git help config               在浏览器打开

git config -h                 在 git 终端打开
```
