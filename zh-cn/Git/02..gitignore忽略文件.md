#### 1. .gitignore 忽略文件

```txt
#                         注释

/node_modules             忽略 node_modules文件夹下所有内容

index.css                 忽略 index.css

*.js                      忽略所有 .js 文件
!test.js                  可以暂存 test.js (即便前面使用了  *.js  )

lib/*.txt                 忽略lib文件夹下近一代的所有 .txt文件
lib/**/*.txt              忽略lib文件夹下所有 .txt文件

...
```
