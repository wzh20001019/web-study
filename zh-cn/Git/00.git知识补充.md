## 1. 查看远程 git 仓库地址

```txt
git remote -v
```

## 2. 储存

```txt
// 存储   git stash save "创建了一条记录"
git stash
```

```txt
// 查看 stash 记录
git stash list
```

```txt
// 删除 stash 记录
git stash drop

// 删除 指定的记录
git stash drop stash@{3}

// 删除所有的stash记录
git stash clear
```

```txt
// 恢复 stash 所存储的内容    git stash apply 默认回复栈顶(也就是 0 )
git stash apply stash@{0}
```
