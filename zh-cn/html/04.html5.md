# Html5 新增语义化标签

```html
<header>头部标签</header>

<nav>导航标签</nav>

<article>内容标签</article>

<section>块级标签</section>

<!-- 在 article 标签之外, 与article内容相关的辅助信息 -->
<aside>侧边栏标签</aside>

<footer>页面尾部标签</footer>
```

# Html5 新增视频(video)标签

```html
<!-- 写法1 -->
<video src="./01.mp4" controls></video>

<!-- 写法2 -->
<video controls>
	<source src="./01.mp4" type="video/mp4" />
	你的浏览器不支持视频播放
</video>
```

```txt
video标签属性:

autoplay              自动播放(Chrome浏览器要添加 muted 解决)

controls              显示播放控件

loop                  循环播放

muted                 静音播放

src                   引入视频文件

preload               auto / none  是否预先加载视频(有 autoplay 属性就忽略)
```

# Html5 新增视频(audio)标签

```html
<!-- 写法1 -->
<audio src="audio/1.mp3" controls></audio>

<!-- 写法2 -->
<audio controls>
	<source src="audio/1.mp3" type="audio/mpeg" />
	你的浏览器不支持音频播放
</audio>
```

```txt
audio标签属性:

autoplay              自动播放(Chrome浏览器不支持)

controls              显示播放控件

loop                  循环播放
```

```html
<!-- 拓展   外部引入 -->

<embed src="audio/1.mp3" width="0" height="0" type="" />

<object data="audio/1.mp3" type=""></object>
```
