# 表格

#### 1. 表格

```txt
<caption>表格标题标签</caption>

rowspan     跨行(竖)合并单元格
colspan     跨列(横)合并单元格
```

```css
/* 单元格相邻边框合并 */
table {
	border-collapse: collapse;
}
```

# 列表

#### 1. 无序列表

```html
<!--  diso: 默认小黑圆点,  circle: 空心圆点,  square: 黑色正方块  -->
<ul type="diso">
	<li>有序列表</li>
</ul>
```

#### 2. 有序列表

```html
<!-- 1: 123,   a: abc,   A: ABC,   I: I II III -->
<ol type="1">
	<li>无序列表</li>
</ol>
```

#### 3. 定义列表

```html
<dl>
	<dt>名称</dt>
	<dd>描述</dd>
</dl>
```

# 表单

#### 1. 常见表单标签

```html
<form></form>

<input name="name" value="value" />

<!-- 下拉列表 -->
<select>
	<option value="">1</option>
	<option value="">2</option>
</select>

<textarea>文本域</textarea>
```

#### 2. label 与 input 标签配合使用

```html
<label for="app">123</label> <input type="text" id="app" />
```

#### 3. input 标签

```html
<!-- input 必须有 name 属性 -->
<form>
	<input type="text" name="name" placeholder="默认显示文本" maxlength="最多输入字符" />
	<input type="password" />
	<input type="submit" value="提交" />
	<input type="reset" value="重置" />
	<input type="button" value="普通按钮" />
	<input type="radio" name="only" checked="checked" />单选框(多个 name属性值一样checked:默认选中)

	<input type="checkbox" />复选框(多个)

	<!-- 上传文件  accept(指定上传文件格式)  multiple(可多选文件) -->
	<input type="file" accept="image/jpg, image/png" multiple />
</form>
```
