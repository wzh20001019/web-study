#### 2. 常用的浏览器及内核

```txt
1 IE浏览器: Trident

2、火狐浏览器(Firefox):  Gecko

3、谷歌浏览器(Chrome):  Blink(新版) & Webkit(旧版)

4、苹果浏览器(Safari)：Webkit内核

5、Opera浏览器内核：Blink
```

#### 3. 浏览器私有前缀

```txt
-webkit-          Chrome, safari(苹果)

-moz-             firefox(火狐)

-ms-              IE浏览器

-o-               Opera
```

#### 4. DOCTYPE

```txt
<!DOCTYPE html>  文档类型声明

<!DOCTYPE html>: 当前页面采取的是 HTML5 版本来显示网页
```
