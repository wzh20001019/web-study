#### 1. process 进程

```javascript
// 读取进程
console.log(process.argv)
```

#### 2. 运行脚本(package.json)

```json
"scripts": {
    // npm run test      相当于执行: node index.js
    "test": "node index.js",
    "check": "node -v",

    // & 并行 (同时执行多个文件)
    "parallel": "node main.js & node main2.js",

    // && 从前至后执行
    "order": "node main.js && node main2.js"
  },
```

#### 3. cross-env 设置不同的环境变量

```txt
npm install --save-dev cross-env            安装
```
