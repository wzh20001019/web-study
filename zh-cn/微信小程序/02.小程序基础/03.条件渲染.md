#### 条件渲染

```txt
// wx:if     wx:elif      wx:else
<view wx:if="{{ count === 1 }}">1</view>              wx:if=""

<view wx:elif="{{ count === 2 }}">2</view>            wx:elif=""

<view wx:else>其他</view>                             wx:else
```

```txt
// hidden     true: 隐藏内容   false: 显示内容
<view hidden="{{ true }}">123</view>
```
