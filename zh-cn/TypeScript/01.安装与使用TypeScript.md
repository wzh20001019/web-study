#### 全局安装 TypeScript

```txt
npm i -g typescript
```

#### 检查安装

```txt
tsc -v
```

#### 转换成 js 文件

```txt
tsc ts文件名
```

#### VScode 中自动编译 ts 文件

###### 1. 生成配置文件 tsconfig.json

```txt
tsc --init
```

###### 2. 修改 tsconfig.json 配置

```txt
"outDir": "./js",        生成js文件到 js文件夹

"strict": false          不开启严格模式
```

###### 3. 启动监视任务:

```txt
tsc
```
