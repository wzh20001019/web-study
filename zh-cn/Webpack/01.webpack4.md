#

## webpack4 相关包

```text
// 全局安装 webpack4
npm install webpack@4.46.0 webpack-cli@3.3.12 -g

// 本地安装(项目中使用) webpack4
yarn add webpack@4.46.0 webpack-cli@3.3.12 -D

// css-loader
yarn add style-loader@1.3.0 css-loader@3.6.0 -D

// less-loader
yarn add less-loader@5.0.0 -D

// 打包 html 文件   需要在plugin中配置
yarn add html-webpack-plugin@4.5.2 -D

// 处理图片文件
yarn add url-loader file-loader -D

// 处理 html 中的图片, 负责引入img, 从而能够被 url-loader处理
yarn add html-loader@1.3.2 -D
```
