#### 抽象语法树 和 虚拟节点的关系

```txt
模板语法 --> AST 抽象语法树 --> 渲染函数(h 函数) --> 虚拟节点 --> 界面
```

#### h 函数(渲染函数)

```h
h('div', {attrs: { className: 'box' }}, [
    h('h3', '我是一段标题'),
    h('ul', [
        h('li', 1),
        h('li', 2),
        h('li', 3)
    ])
])
```

#### AST(Abstract syntax tree) 抽象语法树 (本质是一个 js 对象)

```html
<div class="box">
    <h3 class="title">我是一段标题</h3>
    <ul>
        <li v-for="(item, index) in arr" :key="index">{{item}}</li>
    </ul>
</div>
```

↓↓↓↓↓↓↓ (Vue 中已字符串视角 解析成 AST)

```json
// tag: 标签名      attrs: 属性      type: 节点类型      children : 字元素

{
    "tag": "div",
    "attrs": [{ "name": "class", "value": "box" }], // class 类, 类名为 box
    "type": 1, //1: 元素节点, 2: 属性节点, 3: 文本节点
    "children": [
        //子元素
        {
            "tag": "h3",
            "attrs": [{ "name": "class", "value": "title" }],
            "type": 1,
            "children": [{ "text": "我是一段标题", "type": 3 }]
        },
        {
            "tag": "ul",
            "attrs": [],
            "type": 1,
            "children": [
                {
                    "tag": "li",
                    "attrs": [],
                    "for": "arr",
                    "key": "index",
                    "alias": "item",
                    "type": 1,
                    "children": []
                }
            ]
        }
    ]
}
```
